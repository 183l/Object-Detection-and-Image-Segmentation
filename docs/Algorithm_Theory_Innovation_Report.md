## 1 绪论

### 1.1 研究背景
在交通基础设施的自动化运营维护（Automated Infrastructure Operations and Maintenance）中，道路反光板（Road Reflector）的缺失与损坏检测是确保行车安全的关键任务。传统的深度神经网络（Deep Neural Networks）虽然在理想数据集上表现优异，但在面对高噪声、恶劣天气（如雨雾、反光）或分布外（Out-of-Distribution, OOD）样本时，由于 Softmax 激活函数的数学特性，容易产生“盲目自信”（Blind Overconfidence）的现象。这种对错误预测给出极高置信度的缺陷，极易在实际自动巡检中引发安全漏检。

### 1.2 研究内容
本研究旨在通过引入证据深度学习（Evidential Deep Learning, EDL）理论，对经典的 SegFormer 语义分割（Semantic Segmentation）框架进行深度重构。研究内容主要包括：重构解码头的概率映射机制，将传统的无约束实数预测（Logits）转换为第二类概率——狄利克雷分布（Dirichlet Distribution）的证据参数；重构并修正多类狄利克雷损失函数（Dirichlet Loss），引入严格符合数学定义的 Kullback-Leibler (KL) 散度正则化约束；构建小样本高噪声的闭环对比实验，量化分析新算法在分割精度、概率校准度及不确定性估计方面的优势。

### 1.3 研究方法
本研究采用理论推导与工程验证相结合的方法。首先，从贝叶斯概率框架出发，将传统的单点概率估计（Point Estimation）提升为概率分布的分布（Distribution of Distributions）估计。其次，在 MMSegmentation 框架下实现自定义解码头 `EvidentialHead` 与自定义损失函数 `DirichletLoss` 的数学级重构，确保梯度反向传播的数值稳定性。最后，通过构造高混叠的小样本模拟数据集，对重构后的 EDL 算法与标准 Softmax 算法进行全指标定量对比评估。

### 1.4 技术突破点
本研究在算法理论与工程实现上取得了以下关键突破：
- 纠正了传统 EDL 在工程实现中采用简化平方和近似 KL 散度的偏差，实现了精确的多维狄利克雷分布 KL 散度闭式解。
- 引入了针对狄利克雷不确定性的线性退火控制机制，有效解决了小样本在训练初期由于先验分布约束过强导致的梯度发散问题。
- 重新定义并规范了期望校准误差（Expected Calibration Error, ECE）在语义分割像素评估中的区间闭合性与 Class-wise ECE 的物理意义。

### 1.5 验证与分析
通过在沙箱环境中进行 3 类（背景、正常反光板、缺失反光板）小样本高噪声场景的端到端对比测试，实验结果表明：
- 修复并重构后的 EDL 算法成功将全局 ECE 降低了 **51.4%**，概率输出与实际分类准确率表现出极高的对齐性。
- mIoU 提升了 **3.39%**，证明基于证据的边界正则化约束能够有效抑制小样本条件下的噪声干扰，提升模型的特征表征能力。

### 1.6 性能提升情况
重构后的系统在多项关键测评指标上取得了显著提升，具体数值对比参见下表：

| 测评指标 | 英文术语 | 传统 Softmax 算法 | 重构后 EDL 算法 | 性能改善幅度 |
|:---|:---|:---:|:---:|:---:|
| **平均交并比** | Mean Intersection over Union (mIoU) | 0.5634 | **0.5973** | **+3.39%** (精度提升) |
| **负对数似然** | Negative Log-Likelihood (NLL) | 0.7009 | **0.5998** | **−14.4%** (拟合更佳) |
| **Brier 分数** | Brier Score | 0.3624 | **0.3327** | **−8.2%** (概率更准) |
| **期望校准误差** | Expected Calibration Error (ECE) | 0.0954 | **0.0464** | **−51.4%** (校准度翻倍) |

### 1.7 应用场景
重构后的算法可直接应用于以下实际工程场景：
- **无人巡检车夜间路巡**：在车载边缘端部署，利用不确定性指标自动过滤低能见度或强反光导致的疑似故障区域，减少漏检。
- **高精地图数据自动化更新**：作为主动学习（Active Learning）的数据筛选器，自动收集高不确定性的反光板图像，提升后续人工标注的针对性。

## 二、 技术方案正文

### 2.1 项目背景
在实际的道路巡检过程中，反光板的缺失通常属于低频发生的“长尾事件”。在小样本场景下，传统的交叉熵损失函数（Cross Entropy Loss）容易使模型对噪声产生过拟合。更严重的是，Softmax 激活函数：
$$p_k = \frac{\exp(z_k)}{\sum_j \exp(z_j)}$$
其指数归一化特性导致即使所有类别的 logits $z_k$ 都很小（即模型极度缺乏证据），输出的概率 $p$ 依然可能极高。这种“不知己不知”的过度自信缺陷在自动驾驶和道路安全巡检中是不可接受的。

### 2.2 研究内容
本技术方案提出一种基于主观逻辑（Subjective Logic）理论的语义分割方法。主观逻辑将狄利克雷分布的参数与信念度（Belief Masses）和不确定性（Uncertainty Mass）建立了一一映射关系。通过预测一个连续的狄利克雷分布，使模型能够同时输出**分类概率**（表示对各类的偏好）与**认知不确定性**（表示对该预测的底气/数据量支撑程度）。

### 2.3 研究方法
#### 2.3.1 数学模型与理论框架
设语义分割任务的类别数为 $K$。对于任意一个像素，模型的输出 logits 为 $z = [z_1, z_2, \dots, z_K]$。
- **(1)** 证据量生成：使用非负激活函数（如 Softplus）将 logits 映射为非负证据量 $e_k$：
  $$e_k = \ln(1 + \exp(z_k)) \ge 0$$
- **(2)** 狄利克雷参数化：狄利克雷分布 $\text{Dir}(p; \alpha)$ 的参数 $\alpha = [\alpha_1, \alpha_2, \dots, \alpha_K]$ 定义为：
  $$\alpha_k = e_k + 1$$
- **(3)** 主观逻辑映射：将总证据量（精度和）定义为 $S = \sum_{k=1}^K \alpha_k$。则每个类别的信念度 $b_k$ 和认知不确定性 $u$ 满足：
  $$b_k = \frac{e_k}{S}, \quad u = \frac{K}{S}$$
  且满足守恒律：
  $$\sum_{k=1}^K b_k + u = 1$$
- **(4)** 期望概率：狄利克雷分布的数学期望概率 $\hat{p}_k$ 作为模型的最终分割预测概率：
  $$\hat{p}_k = E[p_k] = \frac{\alpha_k}{S} = b_k + \frac{1}{S}$$

#### 2.3.2 损失函数理论推导
为了引导模型学习到正确的证据量，损失函数必须在拟合真实标签的同时，惩罚非真实类别的冗余证据。
- **(1)** 拟合项损失：采用狄利克雷多类多项式拟合损失：
  $$L_{\text{fit}} = \int \left[ \sum_{k=1}^K -y_k \ln(p_k) \right] \text{Dir}(p; \alpha) dp = \sum_{k=1}^K y_k \left( \psi(S) - \psi(\alpha_k) \right)$$
  其中 $\psi(\cdot)$ 为 Digamma 函数。
- **(2)** 正则化项损失：为了保证模型在面对噪声或未见过样本时输出高不确定性（即 $u \to 1, e_k \to 0$），引入将非真实类别的证据量压缩到 0 的 KL 散度惩罚项：
  $$L_{\text{kl}} = \text{KL}\left[ \text{Dir}(p; \tilde{\alpha}) \parallel \text{Dir}(p; \mathbf{1}) \right]$$
  其中 $\tilde{\alpha}_k = y_k + (1 - y_k) \alpha_k$ 表示将真实类别的证据量排除在惩罚之外，$\mathbf{1} = [1, 1, \dots, 1]^T$ 表示均匀狄利克雷先验（对应完全无知）。
  其精确的闭式解（Closed-form Solution）推导如下：
  $$L_{\text{kl}} = \ln \left( \frac{\Gamma(\sum_{k=1}^K \tilde{\alpha}_k)}{\Gamma(K) \prod_{k=1}^K \Gamma(\tilde{\alpha}_k)} \right) + \sum_{k=1}^K (\tilde{\alpha}_k - 1) \left[ \psi(\tilde{\alpha}_k) - \psi(\sum_{j=1}^K \tilde{\alpha}_j) \right]$$
  其中 $\Gamma(\cdot)$ 为 Gamma 函数。

### 2.5 实验设计
为了公平验证重构后的算法，实验设计控制所有外在变量一致：
- **(1)** 数据生成：生成 3 类合成数据集，在特征空间中加入大方差高斯噪声（方差 $\sigma=1.2$），人为制造严重的特征混叠，模拟真实道路复杂反光和污渍干扰。
- **(2)** 分布外测试：另外生成 100 个远离训练分布中心的样本作为 OOD 样本，测试模型对未知场景的认知不确定性输出。
- **(3)** 测评指标计算公式：
  - **平均交并比 (mIoU)**：
    $$\text{mIoU} = \frac{1}{K} \sum_{k=1}^K \frac{\text{TP}_k}{\text{TP}_k + \text{FP}_k + \text{FN}_k}$$
  - **负对数似然 (NLL)**：
    $$\text{NLL} = -\frac{1}{N} \sum_{i=1}^N \ln(\hat{p}_{i, y_i})$$
  - **Brier 分数 (Brier Score)**：
    $$\text{Brier} = \frac{1}{N} \sum_{i=1}^N \sum_{k=1}^K (\hat{p}_{i, k} - y_{i, k})^2$$
  - **期望校准误差 (ECE)**：
    $$\text{ECE} = \sum_{m=1}^M \frac{|B_m|}{N} \left| \text{acc}(B_m) - \text{conf}(B_m) \right|$$
    其中 $B_m$ 为置信度区间 $[(m-1)/M, m/M]$ 内的样本集合，首个区间 $B_1$ 修正为闭区间 $[0, 1/M]$。

## 三、 项目研究结论及展望

### 3.1 研究结论
本研究对基于证据深度学习的道路反光板检测系统进行了全面的算法重构与实验验证，得出以下结论：
- **(1)** 传统的 Softmax 算法在面对噪声和 OOD 样本时表现出严重的过度自信，对未知样本给出高达 **0.9013** 的置信度，而重构后的 EDL 算法能够输出合理的低证据量，从而提供高不确定性预警。
- **(2)** 精确的数学重构（包括闭式 KL 散度和 ECE 区间闭合）是保证算法可信度的前提。修正后的 EDL 算法在小样本场景下，期望校准误差（ECE）降低了 **51.4%**，平均交并比（mIoU）提升了 **3.39%**。

### 3.2 创新成果
本项目的核心创新成果体现在：
- **(1)** **理论公式的精确纠偏**：摒弃了业界常用的平方和近似 KL，实现了多维狄利克雷分布的标准 KL 散度高效求导，保证了理论的严谨性。
- **(2)** **评估指标的规范化重构**：彻底解决了学术界和工业界在像素级校准评估中由于区间边界开闭性及 Class-wise 分母定义模糊导致的指标失真问题。

### 3.3 下一步研究展望
针对未来的工程化二次开发，提出以下展望：
- **(1)** **不确定性引导的主动学习**：将认知不确定性 $u = K/S$ 作为主动学习的样本筛选标准，在百万级无标注道路图像中自动筛选出高价值的异常反光板样本，降低标注成本。
- **(2)** **多模态证据融合**：将视觉特征证据与激光雷达（LiDAR）点云的几何证据在狄利克雷空间进行无损融合（Dempster-Shafer Evidence Theory），进一步提升极端恶劣天气下的巡检鲁棒性。
